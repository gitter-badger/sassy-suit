    typeSet 

    @function typeSet($typeBase, $typeIndexBegin, $typeIndexEnd, $typeSizeRatio, $typeLineHeightRatio, $typePaddingRatio, $typeLetterSpacingRatio) { ... }
    View source

    Description

    Core typography function, which validates and calculates base values for:

    • font-size
    • line-height
    • letter-spacing
    • padding

    Has option to round selected values

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $typeBase

    core value

    		 Number — none
    $typeIndexBegin

    list's starting point

    		 Number — none
    $typeIndexEnd

    list's last point

    		 Number — none
    $typeSizeRatio

    font-size ratio value

    		 Number — none
    $typeLineHeightRatio

    line-height ratio value

    		 Number — none
    $typePaddingRatio

    padding ratio value

    		 Number — none
    $typeLetterSpacingRatio

    letter-spacing ratio value

    		 Number — none

    Returns

    List

    Throws

    • $typeBase - is not a measure!

    • $typeSizeRatio - is not a positive integer or float, which is more than 1!

    • $typeLineHeightRatio - is not positive and more than 1!

    • $typeLetterSpacingRatio - is not a positive integer or float, which is more than 1!

    • $typeLetterSpacingRatio - is not measure or equals 0!

    • $typePaddingRatio - is not measure or equals 0!

    • $typeIndexBegin - is not measure or equals 0!

    • $typeIndexBegin - is not 0 or a negative integer!

    • $typeIndexEnd - is not a positive integer!

    Used by

    • getTypeSetResponsive
    • typeSet

    Links

    • https://github.com/SassySuit/sassySuit/blob/master/tools/type/_typeSetFunction.scss

    typeSetResponsiveRatios 

    @function typeSetResponsiveRatios($value) { ... }
    View source

    Description

    Typography function, which validates, generates and calculates responsive ratios for each typography break point.

    Number of ratios - $typeBpsRatios has to be eqaul to a number of typography breakpoints - $typeBps.

    Each value of $typeBpsRatios has to match every value of $typeBps.

    If $typeBpsRatios is a single number and $typeBps is a list, than typeSetResponsiveRatios function will generate a list with length($typeBps) length and $value values in it.

    Parameters

    parameter Name parameter Description parameter Type parameter Default value
    $value

    responsive ratios or ratio

    		 Number, List — none

    Returns

    List, Number

    Throws

    • $typeBpsRatios - is not a list with positive numbers or a number

    • typeSetResponsive
